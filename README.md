# SABORÉ RESTAURANTE

Sistema web desarrollado para la gestión de reservas de un restaurante.

## Tecnologías utilizadas

* React
* Vite
* JavaScript
* HTML5
* CSS3
* MySQL
* Node.js
* Express

## Funcionalidades

* Página principal del restaurante
* Información sobre el restaurante
* Menú digital
* Registro e inicio de sesión
* Reservación de mesas
* Gestión de cuenta de usuario
* Visualización de reservas
* Panel administrativo (en desarrollo)
* Integración con base de datos MySQL (en desarrollo)

## Instalación

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```
